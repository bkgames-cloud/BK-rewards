# Fichiers audio pour les effets sonores

Ce dossier contient les fichiers audio utilisés pour les effets sonores de l'application.

## Fichiers requis

- `coin.mp3` : Son de pièces (coin sound) joué quand l'utilisateur gagne un ticket
- `click.mp3` : Son de clic (pop) joué sur les boutons principaux

## Sources recommandées

Vous pouvez trouver des sons gratuits sur :
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## Format

- Format : MP3
- Taille recommandée : < 50 KB par fichier
- Durée : 0.5 - 1 seconde pour les sons de clic, 1-2 secondes pour les sons de pièces

## Exemples de recherche

- Pour `coin.mp3` : "coin drop", "money sound", "coins falling"
- Pour `click.mp3` : "button click", "pop sound", "ui click"
